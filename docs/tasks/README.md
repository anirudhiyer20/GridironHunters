# Task Notes

Use one markdown file per task or chat thread that produces a meaningful outcome.

Suggested naming:

- `001-repo-setup.md`
- `002-supabase-setup.md`
- `003-vscode-workspace.md`

Each task note should include:

## Goal

What we were trying to do.

## Decisions

What was chosen.

## Commands Run

Important commands only.

## Outcome

What is now true.

## Next Step

What should happen next.
