# MVP Scope And Release Shape

## Decision

Gridiron Hunters MVP will include auth, leagues, draft, capture battles, gauntlet PvP, playoffs, bots, and admin tools.

The initial release will be:

- closed beta
- web app first
- invite-only via reusable codes
- in-app notifications only

## Why

This scope covers the full seasonal gameplay loop without taking on native mobile, licensed branding, or broader distribution too early.

## Consequences

- The web experience must be strong enough to serve as the primary product
- Operational tooling matters early because the beta is invite-driven and admin-supported
- Features like iOS, waivers, and traded picks stay out of scope until the MVP loop is proven
