# Asset Source Ledger

Use this ledger for every external visual/audio/font asset added to the repo.

## Rules
- No external asset may be merged without a ledger entry.
- Record the exact source URL and license text/location.
- If license is unclear, do not use asset until clarified.
- Prefer assets with explicit permissive licenses (MIT, CC0, CC-BY with attribution plan).
